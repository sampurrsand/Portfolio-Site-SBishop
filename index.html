My Work

Evermore

Evermore is a responsive web app for an online storefront that sells wedding supplies. Over the course of one month, I created the brand guidelines, made low-fidelity wireframes, mapped out a user flow diagram, gathered feedback on a mid-fidelity prototype, and designed the UI for mobile, tablet, and desktop screens.

Serentripity

Serentripity is a responsive, location-based recommendation web app. Its purpose is to recommend users destinations, things to do, and places to stay based on their interests. Over the course of 3 months, I gathered competitive and user research, mapped out a user flow diagram, created wireframes, conducted usability testing, and designed several iterations of the UI for mobile, tablet, and desktop screens.
